# Photo Viewer v1.0.1 설치판

- 앱과 Windows 파일/제품 버전: `1.0.1` / `1.0.1.0`
- 실행 파일: `PhotoViewer.exe`
- 설정: `%LocalAppData%\Photo Viewer\settings.json`
- 구 `Portable Photo Viewer` 설정은 새 설정이 없을 때 한 번 자동 이전
- 설치 범위: 현재 사용자 또는 모든 사용자
- 등록 후보: 이미지 39종, 영상 제외
- 설치 완료 후 Windows 기본 앱 설정 화면을 자동으로 열지 않음
- 사용자용과 전체 사용자용 중복 설치 차단
- 코드 서명 없음
