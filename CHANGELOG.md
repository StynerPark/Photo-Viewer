# 변경 이력

## v1.0.1

- Retry viewer deletion once after releasing transient image/video worker locks on Windows.
- 제품을 설치형 `Photo Viewer`로 재브랜딩했습니다.
- `PhotoViewer.exe`, `PhotoViewer.Image`와 새 설치/제거 항목을 사용합니다.
- 설정을 사용자 LocalAppData에 저장하고 v1.0.0 설치형 설정을 자동 이전합니다.
- 설치 완료 후 기본 앱 설정 화면 자동 실행을 제거했습니다.
- 39개 이미지 연결 후보 선택, 영상 연결 제외와 설치 범위 중복 차단을 적용했습니다.
- 구·신 기본 앱 등록을 정리하는 별도 도구를 제공합니다.
- v0.2.4~v0.2.9에서 검증한 성능·UI·뷰어·영상 수정 전체를 포함합니다.

## v1.0.0

- 검증된 v0.2.9를 기반으로 사용자/전체 사용자 설치기와 확장자 선택 UI를 추가했습니다.
- 설치형 설정을 사용자 LocalAppData에 분리했습니다.

## v0.2.9

- 영상 전환 시 느린 VLC 정지를 UI 스레드 밖으로 이동하고 다음 영상 시작 지연을 줄였습니다.

## v0.2.8

- 단일·더블·트리플 영상 재생에 저장 가능한 무한 반복 토글을 추가했습니다.

## v0.2.7

- 뷰어에서 파일을 삭제해도 뷰어를 유지하고 다음 미디어를 선택하도록 수정했습니다.

## v0.2.6

- Windows 탐색기에서 연결 파일을 열 때 창을 전면으로 가져오도록 수정했습니다.

## v0.2.5

- 기본 뷰어 모드는 프로그램 시작 때만 적용하고 실행 중 변경한 모드는 탐색기 복귀 후에도 유지하도록 수정했습니다.

## v0.2.4

- v0.2.2 탐색기 트리 구조와 스플리터를 유지하면서 폴더·탭·정렬·썸네일·뷰어 전환을 비동기화했습니다.
- 오래된 작업 취소, 캐시 제한, 웹툰 원본 비율과 빠른 탐색 중 마지막 요청 우선 처리를 추가했습니다.
- 검정/흰 빈 화면, 끝점 재로딩과 직접 실행 후 빈 탐색기 복귀를 수정했습니다.
- UI 버튼, 탭, 메뉴, 설정창과 선택 상태를 통일하고 불필요한 상세 열을 제거했습니다.

## v0.2.2 이하

기존 Git 이력과 GitHub Releases에 v0.1.0부터 v0.2.2까지의 변경과 배포 파일이 보존되어 있습니다.
